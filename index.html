This simulates the old Stanford course selection method over our new proposed method.

The output metrics are explained as follows:

  • Success Rate: Attempted enrollments in non-full classes.
  • Full Rate: Attempted enrollments in full classes.
  • Drop Rate: Percent of slots in enrolled classes that a student chose to drop.
  • Pareto Inefficiency: Percentage of classes that a student could have gotten into that was dropped by someone else had the other student not enrolled in the first place.

For the generate graphs, each column represents the number of filled classes encountered before successfully enrolling per student over all enrollment attempts. Notice the above statistics are similar in the default case, however the distributions in the graphs below differ. The distrbution for the new system tends to be more centralized meaning the number of students that encounter a lot trouble enrolling is lowered at the expense of those who don't encounter any trouble which shows that the new system is more equitable.
